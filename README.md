# SNHU CS360 Portfolio
## Bryan Chan - 2024


### Briefly summarize the requirements and goals of the app you developed. What user needs was this app designed to address?
- This app is designed to be an inventory tracker, which displays the item name and count in a grid view. Within the grid, the user is able to increase and decrease the item count. In order to utilize this feature, the user can either login, create a new account, or utilize the app without creating an account. The entire app uses a SQLite database for the user account and the inventory. This application was designed to help users keep track of their inventory.

### What screens and features were necessary to support user needs and produce a user-centered UI for the app? How did your UI designs keep users in mind? Why were your designs successful?
- Utilize a recycler view was needed for the app's user interface, since it makes the app's contents organized and condensed. This will ensure the design's success, since it seeks to make the interface easier for the user to use.

### How did you approach the process of coding your app? What techniques or strategies did you use? How could those techniques or strategies be applied in the future?
- When developing the app, I started with the front-end, followed by the back-end. I went with this approach, since I wanted to simplify the process by focusing on what the user will focus the most---the interface.

### How did you test to ensure your code was functional? Why is this process important, and what did it reveal?
- The app can be tested for functionality but using a number of use cases to observe its response. 

### Consider the full app design and development process from initial planning to finalization. Where did you have to innovate to overcome a challenge?
- When it came to developing the app, I wasn't sure on how to approach the user interface. However, I eventually went with a Recycler View, since it will provide the grid layout that the app required.

### In what specific component of your mobile app were you particularly successful in demonstrating your knowledge, skills, and experience?
- I was successful in implement the Recycler View for the inventory tracker, since it showed my understanding in constraints, layouts, views, and design.
